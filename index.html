Como usar o RefBox

  1. Selecione a sua marca de câmera.
  2. Arraste referências (poses, luz) para o painel.
  3. O RefBox ajusta cada foto para 3:2 (1920px) automaticamente.
  4. Baixe o ZIP e extraia no cartão SD na pasta:
    DCIM/100MSDCF
  5. Na câmera, use "Recuperar DB de Imagens" se necessário.
