# 📦 RefBox - Studio Reference Manager

O **RefBox** é uma ferramenta leve e prática desenvolvida para fotógrafos que precisam visualizar referências de pose ou iluminação diretamente no LCD da câmera durante o ensaio.

## ✨ Funcionalidades
- **Compatibilidade Total:** Gera arquivos compatíveis com Sony (A7IV, etc), Nikon e Canon.
- **Auto-Ajuste 3:2:** Redimensiona qualquer imagem para a proporção nativa das câmeras, evitando o erro de visualização.
- **Fundo Customizável:** Escolha fundo preto ou branco para preencher as bordas.
- **Geração de ZIP:** Converta várias referências de uma vez e baixe um único pacote organizado.
- **Privacidade:** O processamento é feito localmente no seu navegador. Nenhuma foto é enviada para servidores.

## 🚀 Como usar
1. Acesse: [COLE O SEU LINK DO GITHUB PAGES AQUI]
2. Selecione a marca da sua câmera.
3. Arraste suas imagens de referência.
4. Clique em **Gerar Pacote ZIP**.
5. Extraia os arquivos no seu cartão SD na pasta correspondente (ex: `DCIM/100MSDCF` para Sony).

## 💡 Dica Sony
Se as imagens não aparecerem de imediato, vá no **Menu da Câmera > Reprodução > Recuperar DB de Imagens**.
